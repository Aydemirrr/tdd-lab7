Вариант 7.
Разработать функцию filter_list(items, type), которая принимает список элементов разных типов
(строки, числа) и тип ('str' или 'int'), и возвращает новый список, содержащий только элементы
указанного типа.
Требования к тестам:
- Наличие тестов для фильтрации строк из смешанного списка.
- Наличие тестов для фильтрации чисел из смешанного списка.
- Наличие теста для пустого списка.
- Наличие теста для списка, не содержащего элементов нужного типа.
